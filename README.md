# Mio - Server
RESTful api for Mio-Client 

- [Server Live Link](https://mio-server.herokuapp.com/)
- [Web App Live Link](https://mio-client.vercel.app/)

## Endpoints
- /api/profiles
    - supports GET, POST
- /api/profiles/:id
    - supports GET, PATCH, DELETE
## Technologies
Express.js, PostgreSQL, Node.js
